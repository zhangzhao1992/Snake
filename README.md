# 经典贪吃蛇

一个纯静态网页版贪吃蛇游戏，支持手机、平板和电脑浏览器。

## 功能

- 支持玩家自定义名称。
- 每局结束后自动记录玩家名称和分数。
- 排行榜按分数从高到低排列，展示前 10 名。
- 使用服务器运行时，不同电脑、手机、平板会共享同一个在线排行榜。
- 如果服务器暂时不可用，页面会先使用当前浏览器里的本机记录。

## 本地预览

安装 Node.js 后，在这个文件夹运行：

```powershell
npm start
```

然后访问：

```text
http://127.0.0.1:4173
```

分数会保存到服务器上的 `data/scores.json`。

## 发布到公网

跨设备共享排行榜需要运行 `server.js`，不能只用纯静态托管。入口命令是：

```text
npm start
```

推荐方式：

1. Render / Railway / Fly.io：创建 Node.js Web Service，启动命令填写 `npm start`。
2. VPS / 云服务器：上传整个文件夹，运行 `npm start`，再用域名反向代理到服务端口。
3. 带持久化磁盘的平台：把 `data` 目录放到持久化磁盘中，避免平台重启后丢失排行榜。

发布后，手机、平板、电脑访问同一个网址，就会提交到同一个排行榜。
