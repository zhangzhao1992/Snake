# 经典贪吃蛇

一个纯静态网页版贪吃蛇游戏，支持手机、平板和电脑浏览器。

## 功能

- 支持玩家自定义名称。
- 每局结束后自动记录玩家名称和分数。
- 排行榜按分数从高到低排列，展示前 10 名。
- 分数记录保存在当前浏览器中，刷新页面后仍会保留。

## 本地预览

把这个文件夹作为静态网站打开即可。推荐使用任意静态服务器预览，例如：

```powershell
python -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173
```

## 发布到公网

这是纯静态网站，不需要后端。可直接发布整个文件夹，入口文件是 `index.html`。

推荐方式：

1. Netlify：登录 Netlify 后选择 Add new site，再把整个文件夹拖到部署区域。
2. Vercel：新建项目，导入这个文件夹或仓库，框架选择 Other，输出目录留空。
3. GitHub Pages：把文件提交到仓库，在 Settings > Pages 中选择分支和根目录发布。

发布后，手机、平板、电脑都可以用同一个网址打开游玩。
